# 🏦 Bank Management System – Java Console Application

A simple and powerful **Bank Management System** developed using **Core Java**.
This project simulates real-world banking operations through a console-based application and demonstrates strong understanding of **Object-Oriented Programming, File Handling, Exception Handling, Collections, and Menu-Driven Application Development**.

The application allows users to create accounts, deposit money, withdraw funds, check balances, and maintain transaction records efficiently.

---

# 🚀 Project Objective

The main goal of this project is to build a real-time banking simulation system using Java fundamentals while improving backend logic-building and problem-solving skills.

This project focuses on:

* Core Java programming
* OOP concepts
* Banking operation logic
* User interaction through console
* Data storage using file handling
* Exception handling
* Collection Framework usage
